Description   : Python code. Downloads the audio recording messages you send to a telegram bot ('TOKEN' telegram bot token needs to be added) 
                 and plays back the content in the default audio player of the computer. Audio played in a FIFO queue. 
Use           : Used during a party with music to give the guests the opportunity of broadcasting voice messages to all the audience.
